Projet IoT : Surveillance de Température et Humidité

Application Web Code GitHub

J’ai conçu un système de surveillance de température et d’humidité à distance, basé sur une carte ESP8266 et un capteur DHT11. L’application web a été développée avec Django sur PyCharm, puis déployée sur PythonAnywhere. Le schéma électronique a été réalisé sur EasyEDA, et le circuit imprimé (PCB) usiné à l’aide de FlatCAM. Le boîtier protecteur a été modélisé et imprimé en 3D avec Fusion 360.
